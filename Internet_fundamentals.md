##  Assignment
## 1. What is the Internet?
The Internet is a group of interconnected networks that allows computers and other devices to communicate with each using standardized protocols data.
## 2. Explain the difference between the Internet and the World Wide Web.
The Internet is the global network infrastructure that provides connectivity, while the World Wide Web is a collection of websites and web pages that uses the Internet to deliver content.
## 3. What is the role of an ISP?
An ISP know asInternet Service Provider is a company that provides Internet access.
## 4. What is an IP address?
An IP address is a unique identifier assigned to devices connected to a network.
## 5. What does DNS do?
DNS (Domain Name System) translates domain names into IP addresses.
## 6. Explain the difference between HTTP and HTTPS.
HTTP is used to transfer web pages, whereas HTTPS encrypts communication between users and websites makeing it secure.
## 7. Mention three benefits of the Internet.
 * Instant communication
 * Access to information
 * Online education
## 8. List three Internet security best practices.
 * Use strong passwords
 * Enable two-factor authentication (2FA)
 * Avoid suspicious links
